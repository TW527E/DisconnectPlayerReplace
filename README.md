# Main Features
When the SCP leaves the server, it is automatically replaced by a spectator.

# Default config:
```yaml
s_c_p_replace:
  is_enabled: true
  # When Scp Replace Message
  on_scp_replace: <color=red>{SCPRole}</color> <color=green> is Replaced</color>.
```
